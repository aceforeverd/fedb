WHERE Clause#

Where 子句用于设置过滤条件，查询结果中只会包含满足条件的数据。

Syntax#

WhereClause
         ::= 'WHERE' Expression

SQL语句模版#

SELECT select_expr [,select_expr...] FROM ... WHERE where_condition

边界说明#

在单机版中，所有执行模式均支持WHERE子句。下表说明了集群版各模式的支持情况。
